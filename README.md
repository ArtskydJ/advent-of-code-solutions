# Advent Of Code Solutions

> My solutions to [Advent Of Code](http://adventofcode.com/)...

#### How to use

- install [node.js](https://nodejs.org/en/download)
- clone this repo
- `cd` into this repo
- run `node aoc <day> <half>`
